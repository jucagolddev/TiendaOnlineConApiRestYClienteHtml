JUCASHOP: Proyecto Fullstack

Réplica funcional basada en el e-commerce PcComponentes

Sobre este desarrollo

Esta aplicación web ha sido desarrollada completamente desde cero como una práctica avanzada de programación web. El objetivo principal ha sido replicar la funcionalidad y la estética de una tienda online real sin utilizar frameworks pesados, demostrando un dominio sólido de los lenguajes nativos.

Nota Importante: Esta no es una tienda real. Los productos y precios son ficticios y no se realizan transacciones bancarias reales.

Stack Tecnológico

  • HTML5 SCSS
    Diseño modular, responsive y metodología BEM.
  • JavaScript
    Fetch API, manipulación del DOM y LocalStorage.
  • PHP
    API RESTful personalizada sin frameworks.
  • JSON
    Sistema de base de datos basado en archivos.

Funcionalidades Clave

  • Sistema de Autenticación (Login y sesiones).
  • Carrito de compras con validación de stock en servidor.
  • Carga dinámica de productos vía API.
  • Diseño totalmente Responsive.
  • Arquitectura Cliente-Servidor separada.
